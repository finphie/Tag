# Breakout

Unity製追いかけっこゲームです。

## 説明

Tagは、Unityで開発しているシンプルな追いかけっこゲームです。

## 対応プラットフォーム

### PC

- OS: Windows 7 SP1以降（64bit）
- CPU: x64
- 解像度: 1920x1080以上

## 使い方

### インストール方法

1. [Release](https://github.com/finphie/Tag/releases)から対応したファイルをダウンロード
1. Tag.exeを実行

### アンインストール方法

ダウンロードしたファイルを削除してください。

### 操作方法
